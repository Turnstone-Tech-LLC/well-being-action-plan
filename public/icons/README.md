# PWA Icons

This directory contains Progressive Web App icons in various sizes.

## Required Icon Sizes

- 72x72px
- 96x96px
- 128x128px
- 144x144px
- 152x152px
- 192x192px
- 384x384px
- 512x512px

## Generating Icons

You can use tools like:
- [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [Favicon.io](https://favicon.io/)

## Icon Requirements

- Format: PNG
- Purpose: maskable and any
- Transparent background or solid color
- Simple, recognizable design
- Centered content with safe area padding

## Placeholder

Until custom icons are created, use placeholder images or generate icons from the app logo.
